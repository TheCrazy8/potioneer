Hunger Bens .ben Plugin Generator

Generate Windows plugins for Hunger Bens as .ben files (they are zip archives with a .ben extension). No upload or server required.

What is a valid plugin?
  • Entry file must be one of:
    • plugin.py at the root (default)
    • plugin/__init__.py package entry
    • Exactly one .py file at the root (fallback)
  • Optional API functions inside the entry module:
    • get_custom_content() → dict like { weapons, items, hazards }
    • get_events() → { day: [callables], night: [callables], global: [callables] }
    • get_event_weights() → { callable or name: weight }

    Test your .ben

    1. Copy the downloaded .ben into a recognized plugins folder on Windows. In the GUI, open Settings → Plugins to see folders.
    2. Run the simulator with plugins enabled. Check the log for "[Plugins] Loaded …" and integration messages.
